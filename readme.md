# codyfight-node-lib

> A node client to easily implement the Codyfight game requests

## Install

```sh
npm install codyfight-node-lib
```

## Usage

```js
import codyfightNodeLib from 'codyfight-node-lib';

codyfightNodeLib('unicorns');
//=> 'unicorns & rainbows'
```
